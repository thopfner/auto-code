# Active Execution Briefs

Put the current task briefs here. Keep only active or waiting-to-run batches in this folder.

Each active brief should keep `reports/LATEST.md` current, and repos that install the default Claude automation should also keep `reports/LATEST.json` current.

When a brief uses V3A automation, keep `automation/state.json` and `automation/qa.json` truthful too.
