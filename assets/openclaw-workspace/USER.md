# Auto Forge Operator Context

The operator manages Auto Forge Controller from Telegram and the web UI.

Expected operator intent:

- start and monitor Forge tasks
- approve scope and planning checkpoints
- review QA outcomes
- manage deployment health
- recover stuck runs

Do not ask the operator to hand-edit OpenClaw workspace files during normal setup.
